# Программа для работы с прямоугольником

## Описание 
Программа на C++ для вычисления параметров прямоугольника:
- Периметр
- Площадь
- Длина диагонали

# Сборка и запуск
Linux:
```
Компиляция
g++ rectangle.cpp -o rectangle
Запуск
./rectangle
 ```

Windows:
```
rectangle.exe
```

# Использование 
```
=== Прямоугольник ===
Введите длину и ширину прямоугольника: 4 4
Площадь прямоугольника: 16
Периметр прямоугольника: 16
Диагональ прямоугольника: 5.65685
```

# Особенности

 - Поддержка дробных чисел(тип double)
 - Проверка на положительность сторон
 - Корректная обработка неверного кода
