Программа для работы с треугольником

  Программа на C++ для проверки вычисления периметра треугольника, площади по формуле Герона и определения равнобедренности.

Сборка и запуск
  Linux:
    g++ triangle.cpp -o triangle
    ./triangle

  Windows:
    g++ triangle.cpp -o triangle.exe
    triangle.exe

Использование
  Введите три целых числа — стороны треугольника.
  Пример
    Введите стороны треугольника a, b, c:
    5 5 8
    Периметр: 18
    Площадь по формуле Герона: 36
    Треугольник равнобедренный

Особенности
    Работает только с целыми сторонами.
    Площадь округляется вниз из-за типа int.
    Локализация для русского языка.
