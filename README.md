# Программа для работы с геометрическими фигурами

Программа на c++ для вычисления функций у следующих фигур:

1. Прямоугольник – периметр, площадь, длина диагонали;
2. Треугольник – периметр, площадь по формуле Герона, проверка
на равнобедренность;
3. Трапеция – периметр, площадь, длина средней линии;
4. Окружность – длина окружности, площадь круга, площадь
кругового сектора.


## Структура проекта

 > main.cpp # Главный файл с меню выбора фигуры
 > geometry.h # Заголовочный файл с объявлениями функций
 > rectangle.cpp # Реализация функций для прямоугольника
 > triangle.cpp # Реализация функций для треугольника
 > trapezoid.cpp # Реализация функций для трапеции
 > circle.cpp # Реализация функций для окружности
> README.md # Документация проекта


## Принцип работы:
1. Пользователь запускает программу
2. На экране появляется меню с выбором фигуры
3. Пользователь вводит номер фигуры (1-4) или 0 для выхода
4. Программа запрашивает необходимые параметры
5. Вычисляются и выводятся результаты
6. Программа завершается после выбора 0

## Компиляция всех файлов:
```
g++ main.cpp rectangle.cpp triangle.cpp trapezoid.cpp circle.cpp -o geometry
```

Запуск программы:

На Linux:
./geometry

На Windows:
geometry.exe

# Как работают заголовочные файлы
Файл geometry.h
```
#ifndef GEOMETRY_H
#define GEOMETRY_H

void Rectangle();
void Triangle();
void Trapezoid();
void Circle();

#endif
```

Заголовочный файл содержит только объявления функций. Он подключается во все .cpp файлы, чтобы компилятор знал о существовании функций.


## Файлы реализации(.cpp)
Каждый .cpp файл содержит реализацию конкретных функций:

 > rectangle.cpp - код для прямоугольника
 > triangle.cpp - код для треугольника
 > и т. д.

## Файл main.cpp

Содержит меню и вызывает соответствующие функции в зависимости от выбора пользователя.


# Процесс работы программы

1. Препроцессор: подставляет содержимое geometry.h в main.cpp

2. Компиляция: каждый .cpp компилируется в объектный файл (.o)

3. Линковка: все объектные файлы соединяются в исполняемый файл
