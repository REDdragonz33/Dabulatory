Программа для работы с трапецией на с++
Программа на C++ для вычисления периметра, площади и средней линии трапеции.

Сборка и запуск Linux:

  g++ main.cpp -o main
./main

Windows:

g++ main.cpp -o main.exe
main.exe

Использование

Введите основания: 10 12

Введите боковые стороны: 8 9
 
Введите высоту: 7

Площадь: 77

Средняя линия: 11

Периметр: 39


Особенности

Работает только с целыми сторонами.
Локализация для русского языка.
